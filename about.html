About AfterFiver

This web application supports participants of ISWC2016 who would like to enjoy their stay at KOBE, Japan. It provides three features to enjoy KOBE and around areas based on Linked Open Data, some other Web APIs and multimedia contents.

NOTICE

  • This application collects anonymous usage logs which will be used for research purpose.
  • If you are an Android user, please use Chrome instead of the default browser of Android.

Please tap the QR code below or read it by a barcode reader on your smartphone or tablet to use the application.

Features

To find good place for enjoying time after conference.

To capture Japanese words for enjoying Japanese foods.

To know the latest information in Japan.
